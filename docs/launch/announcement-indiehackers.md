## Post Title

Launched my first digital product: a $19 PWYW UTM Builder template for Google Sheets

## Post Body

### What I built

A Google Sheets template that auto-generates UTM-tagged URLs, standardizes campaign naming conventions, and includes a GA4 channel mapping reference so your traffic stops showing up as "Other."

### Why I built it

I work in marketing and kept seeing the same problems at every company and client:

- Teams using inconsistent UTM parameters (`Social` vs `paid-social` vs `social-media`)
- No shared naming convention for campaigns, making cross-channel reporting a mess
- Traffic miscategorized in GA4 because nobody remembers which `utm_medium` maps to which channel group

I built a spreadsheet to fix it for my own work, then cleaned it up and turned it into a product.

### What's inside

5 tabs in a single Google Sheets/Excel file:

1. **UTM Builder** — dropdown menus for source/medium, auto-generated tagged URLs
2. **Campaign Naming Convention** — formula, 6 rules, 14 channel-specific examples
3. **GA4 Channel Mapping** — the full medium-to-channel reference
4. **Bulk URL Builder** — generate tagged URLs for 10+ channels from one base URL
5. **Quick Start Guide** — 5 minutes to get going

### Pricing

$19 pay-what-you-want on Gumroad. I went with PWYW because the target audience ranges from solo freelancers to agency teams — wanted to keep it accessible while still valuing the work.

### Numbers so far

- Revenue: $0 (just launched)
- Build time: ~2 weeks of evenings
- Stack: Google Sheets + Gumroad + Astro site for discovery
