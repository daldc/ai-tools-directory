## Production Guidelines

### Screenshot Quality:
- Take screenshots at 2x resolution then scale down
- Use clean, realistic sample data
- Ensure fonts are crisp and readable
- Consistent Excel/Sheets theme

### Text Overlays:
- Use brand fonts or professional sans-serif
- High contrast (white text on dark overlay, or dark text with light background)
- Readable at thumbnail size
- Consistent styling across all images

### Branding:
- Use consistent color scheme
- Include subtle branding if desired
- Professional, trustworthy appearance
- Avoid cluttered design

### File Optimization:
- Optimize for web without quality loss
- PNG for text/UI elements, JPG for photos
- Keep file sizes reasonable for fast loading
- Test at various sizes (thumbnail to full view)
