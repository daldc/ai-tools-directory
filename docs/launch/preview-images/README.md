# Preview Images — UTM Builder Gumroad Listing

Place screenshot/preview image files in this directory. The filenames below correspond to the images needed for the Gumroad listing.

## Required Images

| # | Filename | Description | Notes |
|---|----------|-------------|-------|
| 1 | `01-hero-thumbnail.png` | Product mockup — template on a laptop screen, UTM Builder tab visible. Title overlay: "UTM Builder & Campaign Naming Convention Template" | **This is the Gumroad thumbnail.** Use `products/utm-builder-template/assets/product-preview.html` to generate. |
| 2 | `02-tab-overview.png` | Screenshot showing all 5 tab names at the bottom of the spreadsheet | Open the .xlsx in Google Sheets and screenshot the tab bar. |
| 3 | `03-utm-builder-tab.png` | Full screenshot of the UTM Builder tab with example data filled in | Fill in 5–8 example URLs with realistic campaign data. |
| 4 | `04-naming-convention.png` | Screenshot of the campaign naming convention framework with channel examples | Capture the full rules section and at least 6 channel examples. |
| 5 | `05-ga4-mapping.png` | Screenshot of the GA4 channel mapping reference | Show the full medium-to-channel mapping table. |
| 6 | `06-bulk-builder.png` | Screenshot of the Bulk URL Builder with multiple channels populated | Fill in a base URL and show 5+ channels with generated URLs. |

## Image Specs

- **Dimensions:** 1280x720 or 1600x900 (16:9 aspect ratio)
- **Format:** PNG (preferred) or JPG
- **Image 1** is used as the Gumroad thumbnail — make it count
- Keep screenshots clean: no browser chrome, no personal data, use realistic but fictional campaign names

## How to Capture

1. Open `products/utm-builder-template/assets/product-preview.html` in Chrome for the hero image
2. Open the `.xlsx` in Google Sheets for tab screenshots
3. Use a screenshot tool that supports custom dimensions (e.g., CleanShot, Skitch, or Chrome DevTools device toolbar)
4. Crop to 16:9 and export at 2x for retina clarity
