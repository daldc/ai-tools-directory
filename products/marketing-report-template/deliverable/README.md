# Marketing Report Template — User Guide

Thanks for purchasing the Marketing Report Template! This guide will walk you through setup, customization, and best practices for getting the most out of your template.

---

## Quick Start (5 minutes)

1. **Open the template** — Import `Marketing-Report-Template.xlsx` into Google Sheets (File → Import) or open in Excel
2. **Go to the Setup & Instructions tab** — Review metric definitions and configuration options
3. **Set your reporting period** — Update the date fields on the Executive Summary tab
4. **Enter your channel data** — Fill in the Channel Performance tab with your platform data
5. **Review auto-calculations** — KPI Dashboard and trend formulas update automatically
6. **Write your narrative** — Use the Executive Summary prompts and Recommendations framework
7. **Present or export** — Share the sheet, export as PDF, or copy into your preferred presentation format

## Tab-by-Tab Guide

### 1. Executive Summary

This is your "one-page brief" — the tab stakeholders see first.

**How to fill it in:**
- Update the **reporting period** (e.g., "March 2026" or "Q1 2026")
- Enter **top-line KPIs** in the summary metrics section (total spend, total revenue, overall ROAS, total conversions)
- Use the **traffic light indicators**: Green (on/above target), Yellow (within 10% of target), Red (below target by >10%)
- Fill in the **narrative prompts**: What went well, what needs attention, key changes from last period
- The **period-over-period comparison** row auto-calculates if you enter current and previous period values

**Pro tips:**
- Keep the narrative to 3-5 bullet points max. Executives skim.
- Lead with wins, then address gaps with solutions (not just problems)
- Use the "Key Actions Taken" section to show proactive work, not just results

### 2. Channel Performance

Detailed performance data across 8 marketing channels.

**Pre-configured channels:**
1. Paid Search (Google Ads, Microsoft Ads)
2. Paid Social (Meta, LinkedIn, TikTok, etc.)
3. Organic Search (SEO)
4. Organic Social (unpaid social media)
5. Email Marketing
6. Display / Programmatic
7. Affiliate / Referral
8. Direct / Other

**How to fill it in:**
- Enter raw metrics from each platform: spend, impressions, clicks, conversions, revenue
- Calculated fields (CTR, CPA, ROAS, conversion rate) auto-populate via formulas
- Use the "Notes" column for context (e.g., "Launched new campaign mid-month")

**Customization:**
- **Rename channels** to match your specific platforms or groupings
- **Hide unused rows** (right-click → Hide row) — don't delete them in case you need them later
- **Add channels** by inserting rows and copying formulas from an existing row

### 3. KPI Dashboard

At-a-glance performance against targets.

**How to fill it in:**
- Set your **target values** in the Target column (do this once, update quarterly or as goals change)
- Enter **actual values** in the Actual column each reporting period
- **Variance** and **% to target** calculate automatically
- Conditional formatting applies automatically: green (≥100% of target), yellow (80-99%), red (<80%)

**Metrics tracked:**
- Total Marketing Spend
- Total Revenue / Pipeline
- Overall ROAS / ROI
- Total Conversions
- Cost Per Acquisition (CPA)
- Website Sessions
- Conversion Rate
- Email Subscribers (net new)
- Customer Acquisition Cost (CAC)
- Lifetime Value (LTV) — if available

### 4. Monthly Trends

12-month rolling performance data for spotting patterns.

**How to fill it in:**
- Enter monthly totals for each KPI across the 12-month columns
- **MoM change** (month-over-month) auto-calculates in the adjacent column
- **YoY change** (year-over-year) requires the same month from the previous year — enter in the YoY row
- Use the **annotations row** to flag major events (campaign launches, algorithm changes, seasonality, etc.)

**Pro tips:**
- This tab is chart-ready — select any row and insert a sparkline or chart
- Use annotations religiously. Without context, trend data is just numbers
- Review 3-month rolling averages to smooth out noise

### 5. Recommendations

The most valuable tab. This is where you turn data into decisions.

**Framework for each recommendation:**

| Field | What to write |
|-------|---------------|
| **Observation** | What did you notice? (e.g., "Paid social CPA increased 34% MoM") |
| **Data Evidence** | Specific numbers supporting the observation |
| **Recommendation** | What should be done? Be specific and actionable |
| **Expected Impact** | Quantify if possible (e.g., "Estimated 15-20% CPA reduction") |
| **Priority** | High / Medium / Low |
| **Owner** | Who's responsible for executing? |
| **Timeline** | When should this be completed? |

**Pro tips:**
- Aim for 5-8 recommendations per report. More than that and nothing gets prioritized.
- Always tie recommendations back to business goals, not just channel metrics
- Include at least one "continue doing" recommendation — not everything needs to change
- Prioritize ruthlessly. If everything is "High," nothing is.

### 6. Setup & Instructions

Reference tab with metric definitions and configuration guidance.

**Contains:**
- Metric definitions glossary (every metric used in the template, defined)
- Conditional formatting rules (how to adjust thresholds)
- Channel customization guide
- Data source reference (where to pull each metric from)
- Tips for presenting the report effectively

## Best Practices

### Monthly Workflow
1. **Day 1-2 of new month:** Pull data from all platforms, enter into Channel Performance and Monthly Trends
2. **Day 2-3:** Review KPI Dashboard against targets, note variances
3. **Day 3-4:** Write Executive Summary narrative and Recommendations
4. **Day 4-5:** Review with team, finalize, and distribute

### Making It Your Own
- Add your company logo to the Executive Summary tab
- Customize the color scheme by updating cell background colors (keep the traffic light system)
- Add tabs for specific deep-dives relevant to your business (e.g., "SEO Detail," "Email Campaigns")
- Create a "template copy" so you always have a clean starting point each month

### Presenting to Stakeholders
- Start with the Executive Summary — always
- Only go deeper into Channel Performance if asked
- Use the Recommendations tab as your call-to-action slide
- Export as PDF for email distribution, keep the live sheet for interactive review

## Troubleshooting

**Formulas not calculating in Google Sheets?**
After importing, you may need to allow Google Sheets to convert formulas. Go to File → Settings → Calculation → set to "On change."

**Conditional formatting not showing?**
Re-apply via Format → Conditional formatting. The rules are documented in the Setup & Instructions tab.

**Need more than 12 months of trend data?**
Insert additional columns in the Monthly Trends tab and extend the formulas.
