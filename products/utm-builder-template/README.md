## Deliverable Contents

The `.xlsx` file contains 5 tabs:

1. **UTM Builder** — Main URL builder with auto-generating tagged URLs, dropdowns for source/medium, room for 50+ URLs
2. **Naming Convention** — Campaign naming framework with rules and 14 channel-specific examples
3. **GA4 Channel Mapping** — Reference for aligning utm_medium with GA4 default channel groups
4. **Bulk URL Builder** — Multi-channel URL generation from a single base URL (10 pre-configured channels)
5. **Quick Start Guide** — Step-by-step instructions, parameter explanations, pro tips

## Packaging for Gumroad

1. Zip the `deliverable/` folder contents together
2. Upload to Gumroad as a digital download
3. Use `gumroad-listing.md` for all copy and settings
4. Take screenshots from `product-preview.html` for listing images
5. Recommended: also screenshot the actual .xlsx opened in Google Sheets for authenticity
