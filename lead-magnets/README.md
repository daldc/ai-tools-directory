## Marketing AI Tools Cheat Sheet (Q1 2026)

**File:** `marketing-ai-cheat-sheet.pdf` (print-ready) / `marketing-ai-cheat-sheet.html` (source)

A curated cheat sheet covering 15 essential AI tools for marketers, organized by category:

- ✍️ **Content Creation & Copy** — ChatGPT, Jasper, Claude, Copy.ai
- 🎨 **Visual & Design** — Canva Magic Studio, Midjourney, Adobe Firefly
- 📊 **SEO & Analytics** — Surfer SEO, Clearscope
- 📱 **Social Media** — Buffer AI, Lately
- 📧 **Email & Automation** — Klaviyo AI, Seventh Sense
- 🎬 **Video & Audio** — Descript, Synthesia

Each tool includes: use case, pricing tiers, and an editorial recommendation.

### How to Update

1. Edit `marketing-ai-cheat-sheet.html`
2. Open in Chrome and print to PDF, or use headless Chrome:
   ```bash
   chrome --headless --print-to-pdf=marketing-ai-cheat-sheet.pdf marketing-ai-cheat-sheet.html
   ```
3. Commit both files

### Usage

Offer as a free download gated behind email signup on the AI Tools Directory landing page.
